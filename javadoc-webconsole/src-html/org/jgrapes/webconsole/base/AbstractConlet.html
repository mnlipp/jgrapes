071/**
072 * Provides a base class for implementing web console components.
073 * In addition to translating events to invocations of abstract
074 * methods, this class manages the state information of a
075 * web console component instance.
076 * 
077 * # Event handling
078 * 
079 * The following diagrams show the events exchanged between
080 * the {@link WebConsole} and a web console component from the 
081 * web console component's perspective. If applicable, they also show 
082 * how the events are translated by the {@link AbstractConlet} to invocations 
083 * of the abstract methods that have to be implemented by the
084 * derived class (the web console component component that provides
085 * a specific web console component type).
086 * 
087 * ## ConsoleReady
088 * 
089 * ![Add web console component type handling](AddConletTypeHandling.svg)
090 * 
091 * From the web console's page point of view, a web console component 
092 * consists of CSS and JavaScript that is added to the console page by
093 * {@link AddConletType} events and HTML that is provided by 
094 * {@link RenderConlet} events (see below). These events must 
095 * therefore be generated by a web console component. With respect to
096 * the firing of the initial {@link AddConletType}, 
097 * the {@link AbstractConlet} does not provide any support. The
098 * handler for the {@link ConsoleReady} must be implemented by
099 * the derived class itself.
100 * 
101 * ## AddConletRequest
102 * 
103 * ![Add web console component handling](AddConletHandling.svg)
104 * 
105 * The {@link AddConletRequest} indicates that a new web console component
106 * instance of a given type should be added to the page. The
107 * {@link AbstractConlet} checks the type requested, and if
108 * it matches, invokes {@link #doAddConlet doAddConlet}. The
109 * derived class generates a new unique web console component id (optionally 
110 * using {@link #generateConletId generateConletId}) 
111 * and a model (state) for the instance. It 
112 * calls {@link #putInSession putInSession} to make the
113 * model known to the {@link AbstractConlet}. Eventually,
114 * it fires the {@link RenderConlet} event and returns a new
115 * {@link ConletTrackingInfo} with the conlet id and the rendered
116 * modes. 
117 * 
118 * The {@link RenderConlet} event provides to the console session
119 * the HTML that represents the web console component on the page.
120 * The HTML may be generated using and thus depending on the web console
121 * component model.
122 * Alternatively, state independent HTML may be provided followed 
123 * by a {@link NotifyConletView} event that updates
124 * the HTML (using JavaScript) on the console page. The latter approach
125 * is preferred if the model changes frequently and updating the
126 * rendered representation is more efficient than providing a new one.
127 * 
128 * ## RenderConletRequest
129 * 
130 * ![Render web console component handling](RenderConletHandling.svg)
131 * 
132 * A {@link RenderConletRequest} event indicates that the web console page
133 * needs the HTML for displaying a web console component. This may be caused
134 * by e.g. a refresh or by requesting a full page view from
135 * the preview.
136 * 
137 * Upon receiving such an event, the {@link AbstractConlet}
138 * checks if it has state information for the web console component id
139 * requested. If so, it invokes {@link #doRenderConlet doRenderConlet}
140 * with the state information. This method has to fire
141 * the {@link RenderConlet} event that provides the HTML to the console.
142 * 
143 * Method {@link #doRenderConlet doRenderConlet} returns the rendered
144 * view mode(s) which are used to updated the conlet tracking information.
145 * 
146 * ## ConletDeleted
147 * 
148 * ![Web console component deleted handling](ConletDeletedHandling.svg)
149 * 
150 * When the {@link AbstractConlet} receives a {@link ConletDeleted},
151 * it updates the information about the shown conlet views. If the
152 * conlet is no longer used in the browser (no views remain),
153 * it deletes the state information from the session. In any case, it
154 * invokes 
155 * {@link #doConletDeleted(ConletDeleted, ConsoleSession, String, Serializable)}
156 * with the state information.
157 * 
158 * ## NotifyConletModel
159 * 
160 * ![Notify web console component model handling](NotifyConletModelHandling.svg)
161 * 
162 * If the web console component display includes input elements, actions 
163 * on these elements may result in {@link NotifyConletModel} events from
164 * the web console page to the web console. When the {@link AbstractConlet}
165 * receives such events, it checks if state information for the 
166 * web console component id exists. If so, it invokes 
167 * {@link #doNotifyConletModel doNotifyConletModel} with the
168 * retrieved information. The web console component usually responds with
169 * a {@link NotifyConletView} event. However, it can also
170 * re-render the complete portelt display.
171 *

173 * -------------------------------
174 * 
175 * In addition, the class provides support for tracking the 
176 * relationship between {@link ConsoleSession}s and the ids 
177 * of web console components displayed in the console session and support for
178 * unsolicited updates.

335    /**
336     * If set to a value different from `null` causes an event
337     * from the given supplier to be fired on all tracked web console
338     * sessions periodically.
339     *
340     * @param interval the refresh interval
341     * @param supplier the supplier
342     * @return the web console component for easy chaining
343     */
344    @SuppressWarnings("PMD.LinguisticNaming")
345    public AbstractConlet<S> setPeriodicRefresh(
346            Duration interval, Supplier<Event<?>> supplier) {
347        refreshInterval = interval;
348        refreshEventSupplier = supplier;
349        if (refreshTimer != null) {
350            refreshTimer.cancel();
351            refreshTimer = null;
352        }
353        updateRefresh();
354        return this;
355    }
356
357    private void updateRefresh() {
358        if (refreshInterval == null || conletIdsByConsoleSession().isEmpty()) {
359            // At least one of the prerequisites is missing, terminate
360            if (refreshTimer != null) {
361                refreshTimer.cancel();
362                refreshTimer = null;
363            }
364            return;
365        }
366        if (refreshTimer != null) {
367            // Already running.
368            return;
369        }
370        refreshTimer = Components.schedule(tmr -> {
371            tmr.reschedule(tmr.scheduledFor().plus(refreshInterval));
372            fire(refreshEventSupplier.get(), trackedSessions());
373        }, Instant.now().plus(refreshInterval));
374    }

386    /**
387     * A default handler for resource requests. Checks that the request
388     * is directed at this web console component, and calls 
389     * {@link #doGetResource}.
390     * 
391     * @param event the resource request event
392     * @param channel the channel that the request was recived on
393     */
394    @Handler
395    public final void onConletResourceRequest(
396            ConletResourceRequest event, IOSubchannel channel) {
397        // For me?
398        if (!event.conletClass().equals(type())) {
399            return;
400        }
401        doGetResource(event, channel);
402    }
403
404    /**
405     * The default implementation searches for a file with the 
406     * requested resource URI in the web console component's class 
407     * path and sets its {@link URL} as result if found.
408     * 
409     * @param event the event. The result will be set to
410     * `true` on success
411     * @param channel the channel
412     */
413    protected void doGetResource(ConletResourceRequest event,
414            IOSubchannel channel) {
415        URL resourceUrl = this.getClass().getResource(
416            event.resourceUri().getPath());
417        if (resourceUrl == null) {
418            return;
419        }
420        event.setResult(new ResourceByUrl(event, resourceUrl));
421        event.stop();
422    }

424    /**
425     * Provides a resource bundle for localization.
426     * The default implementation looks up a bundle using the
427     * package name plus "l10n" as base name. Note that the bundle 
428     * returned for a given locale may be the fallback bundle.
429     * 
430     * @return the resource bundle
431     */
432    protected ResourceBundle resourceBundle(Locale locale) {
433        return ResourceBundle.getBundle(
434            getClass().getPackage().getName() + ".l10n", locale,
435            getClass().getClassLoader(),
436            ResourceBundle.Control.getNoFallbackControl(
437                ResourceBundle.Control.FORMAT_DEFAULT));
438    }
439
440    /**
441     * Returns bundles for the given locales. 
442     * 
443     * The default implementation uses {@link #resourceBundle(Locale)} 
444     * to lookup the bundles. The method is guaranteed to return a 
445     * bundle for each requested locale even if it is only the fallback 
446     * bundle. The evaluated results are cached for the conlet class.
447     *
448     * @param toGet the locales to get bundles for
449     * @return the map with locales and bundles
450     */
451    protected Map<Locale, ResourceBundle> l10nBundles(Set<Locale> toGet) {
452        @SuppressWarnings("PMD.UseConcurrentHashMap")
453        Map<Locale, ResourceBundle> result = new HashMap<>();
454        for (Locale locale : toGet) {
455            @SuppressWarnings("PMD.AvoidInstantiatingObjectsInLoops")
456            ResourceBundle bundle = l10nBundles
457                .computeIfAbsent(getClass(), cls -> new HashMap<>())
458                .computeIfAbsent(locale, l -> resourceBundle(locale));
459            result.put(locale, bundle);
460        }
461        return Collections.unmodifiableMap(result);
462    }
463
464    /**
465     * Provides localizations for the given key for all requested locales.
466     * 
467     * The default implementation uses {@link #l10nBundles(Set)} to obtain
468     * the localizations.
469     *
470     * @param locales the requested locales
471     * @param key the key
472     * @return the result
473     */
474    protected Map<Locale, String> localizations(Set<Locale> locales,
475            String key) {
476        @SuppressWarnings("PMD.UseConcurrentHashMap")
477        Map<Locale, String> result = new HashMap<>();
478        Map<Locale, ResourceBundle> bundles = l10nBundles(locales);
479        for (Map.Entry<Locale, ResourceBundle> entry : bundles.entrySet()) {
480            result.put(entry.getKey(), entry.getValue().getString(key));
481        }
482        return result;
483    }
484
485    /**
486     * Returns the supported locales and the associated bundles.
487     * 
488     * The default implementation invokes {@link #resourceBundle(Locale)}
489     * with all available locales and drops results with fallback bundles.
490     * The evaluated results are cached for the conlet class.
491     *
492     * @return the result
493     */
494    protected Map<Locale, ResourceBundle> supportedLocales() {
495        return supportedLocales.computeIfAbsent(getClass(), cls -> {
496            ResourceBundle.clearCache(cls.getClassLoader());
497            @SuppressWarnings("PMD.UseConcurrentHashMap")
498            Map<Locale, ResourceBundle> bundles = new HashMap<>();
499            for (Locale locale : Locale.getAvailableLocales()) {
500                if ("".equals(locale.getLanguage())) {
501                    continue;
502                }
503                ResourceBundle bundle = resourceBundle(locale);
504                if (bundle.getLocale().equals(locale)) {
505                    bundles.put(locale, bundle);
506                }
507            }
508            return bundles;
509        });
510    }

521    /**
522     * Returns the tracked sessions and conlet ids as map.
523     * 
524     * If you need a particular session's web console component ids, you 
525     * should prefer {@link #conletIds(ConsoleSession)} over calling
526     * this method with `get(consoleSession)` appended.
527     * 
528     * @return the result
529     */
530    protected Map<ConsoleSession, Set<String>> conletIdsByConsoleSession() {
531        return conletInfosByConsoleSession.entrySet().stream()
532            .collect(Collectors.toMap(Entry::getKey,
533                e -> new HashSet<>(e.getValue().keySet())));
534    }

536    /**
537     * Returns the tracked sessions. This is effectively
538     * `conletInfosByConsoleSession().keySet()` converted to
539     * an array. This representation is especially useful 
540     * when the web console sessions are used as argument for 
541     * {@link #fire(Event, Channel...)}.
542     *
543     * @return the web console sessions
544     */
545    protected ConsoleSession[] trackedSessions() {
546        Set<ConsoleSession> sessions = new HashSet<>(
547            conletInfosByConsoleSession.keySet());
548        return sessions.toArray(new ConsoleSession[0]);
549    }

551    /**
552     * Returns the set of web console component ids associated with the 
553     * console session as a {@link Set}. If no web console components 
554     * have registered yet, an empty set is returned.
555     * 
556     * @param consoleSession the console session
557     * @return the set
558     */
559    protected Set<String> conletIds(ConsoleSession consoleSession) {
560        return new HashSet<>(conletInfosByConsoleSession.getOrDefault(
561            consoleSession, Collections.emptyMap()).keySet());
562    }

564    /**
565     * Track the given web console component from the given session. 
566     * This is invoked by 
567     * {@link #onAddConletRequest(AddConletRequest, ConsoleSession)} and
568     * needs only be used if 
569     * {@link #onAddConletRequest(AddConletRequest, ConsoleSession)} 
570     * or {@link #onRenderConletRequest(RenderConletRequest, ConsoleSession)}
571     * is overridden.
572     *
573     * @param consoleSession the web console session
574     * @param conletId the conlet id
575     * @param info the info to be added if untracked. If `null`
576     * a new {@link ConletTrackingInfo} is added.
577     * @return the conlet tracking info
578     */
579    protected ConletTrackingInfo trackConlet(ConsoleSession consoleSession,
580            String conletId, ConletTrackingInfo info) {
581        Map<String, ConletTrackingInfo> infos
582            = conletInfosByConsoleSession.computeIfAbsent(consoleSession,
583                newKey -> new ConcurrentHashMap<>());
584        ConletTrackingInfo result = infos.computeIfAbsent(conletId,
585            key -> Optional.ofNullable(info)
586                .orElse(new ConletTrackingInfo(conletId)));
587        updateRefresh();
588        return result;
589    }

591    /**
592     * Puts the given web console component state in the session using the 
593     * {@link #type()} and the given web console component id as keys.
594     * 
595     * @param session the session to use
596     * @param conletId the web console component id
597     * @param conletState the web console component state
598     * @return the portlweb console componentet state
599     */
600    @SuppressWarnings({ "unchecked", "PMD.AvoidDuplicateLiterals" })
601    protected Serializable putInSession(Session session, String conletId,
602            Serializable conletState) {
603        ((Map<Serializable,
604                Map<Serializable, Map<String, Serializable>>>) (Map<
605                        Serializable, ?>) session)
606                            .computeIfAbsent(AbstractConlet.class,
607                                newKey -> new ConcurrentHashMap<>())
608                            .computeIfAbsent(type(),
609                                newKey -> new ConcurrentHashMap<>())
610                            .put(conletId, conletState);
611        return conletState;
612    }

629    /**
630     * Returns the state of this web console component's type 
631     * with the given id from the session.

639        return Optional.ofNullable(
640            ((Map<Serializable,
641                    Map<Serializable, Map<String, S>>>) (Map<Serializable,
642                            ?>) session)
643                                .computeIfAbsent(AbstractConlet.class,
644                                    newKey -> new HashMap<>())
645                                .computeIfAbsent(type(),
646                                    newKey -> new HashMap<>())
647                                .get(conletId));
648    }

650    /**
651     * Returns all web console component states of this web console 
652     * component's type from the session.
653     *
654     * @param channel the channel, used to access the session
655     * @return the states
656     */
657    @SuppressWarnings("unchecked")
658    protected Collection<S> statesFromSession(IOSubchannel channel) {
659        return channel.associated(Session.class)
660            .map(session -> ((Map<Serializable,
661                    Map<Serializable, Map<String, S>>>) (Map<Serializable,
662                            ?>) session)
663                                .computeIfAbsent(AbstractConlet.class,
664                                    newKey -> new HashMap<>())
665                                .computeIfAbsent(type(),
666                                    newKey -> new HashMap<>())
667                                .values())
668            .orElseThrow(
669                () -> new IllegalStateException("Session is missing."));
670    }

672    /**
673     * Removes the web console component state of the 
674     * web console component with the given id from the session. 
675     * 
676     * @param session the session to use
677     * @param conletId the web console component id
678     * @return the removed state if state existed
679     */
680    @SuppressWarnings("unchecked")
681    protected Optional<S> removeState(Session session, String conletId) {
682        S state = ((Map<Serializable,
683                Map<Serializable, Map<String, S>>>) (Map<Serializable,
684                        ?>) session)
685                            .computeIfAbsent(AbstractConlet.class,
686                                newKey -> new HashMap<>())
687                            .computeIfAbsent(type(), newKey -> new HashMap<>())
688                            .remove(conletId);
689        return Optional.ofNullable(state);
690    }

729    /**
730     * Checks if the request applies to this component. If so, stops 
731     * the event. If the conlet is completely removed from the browser,
732     * removes the web console component state from the 
733     * browser session. In all cases, it calls {@link #doConletDeleted} 
734     * with the state.
735     * 
736     * @param event the event
737     * @param consoleSession the web console session
738     * @throws Exception the exception
739     */
740    @Handler
741    @SuppressWarnings("PMD.SignatureDeclareThrowsException")
742    public final void onConletDeleted(ConletDeleted event,
743            ConsoleSession consoleSession) throws Exception {
744        Optional<S> optConletState = stateFromSession(
745            consoleSession.browserSession(), event.conletId());
746        if (!optConletState.isPresent()) {
747            return;
748        }
749        String conletId = event.conletId();
750        if (event.renderModes().isEmpty()) {
751            removeState(consoleSession.browserSession(), conletId);
752            for (Iterator<Entry<ConsoleSession, Map<String,
753                    ConletTrackingInfo>>> csi = conletInfosByConsoleSession
754                        .entrySet().iterator();
755                    csi.hasNext();) {
756                Map<String, ConletTrackingInfo> infos = csi.next().getValue();
757                infos.remove(conletId);
758                if (infos.isEmpty()) {
759                    csi.remove();
760                }
761            }
762            updateRefresh();
763        } else {
764            trackConlet(consoleSession, conletId, null)
765                .removeModes(event.renderModes());
766        }
767        event.stop();
768        doConletDeleted(event, consoleSession, event.conletId(),
769            optConletState.get());
770    }

811            consoleSession.browserSession(), event.conletId());

1003         * @param modes the modes
1004         * @return the conlet tracking info
1005         */

1007            if (modes.contains(RenderMode.Preview)) {
1008                renderedAs.add(RenderMode.Preview);
1009            }
1010            if (modes.contains(RenderMode.View)) {

1020         * @return the conlet tracking info

1023            renderedAs.removeAll(modes);
1024            return this;
1025        }

1030        }

1035                return true;
1036            }
1037            if (obj == null) {
1038                return false;
1039            }

1041                return false;

1047                }

1137    /**
1138     * Send to the web console page for adding or updating a complete web 
1139     * console component representation.
1140     */
1141    public static class RenderConletFromReader extends RenderConlet {
1142
1143        private final Future<String> content;
1144
1145        /**
1146         * Creates a new event.
1147         *
1148         * @param request the request
1149         * @param conletType the conlet type
1150         * @param conletId the id of the web console component
1151         * @param contentReader the content reader
1152         */
1153        public RenderConletFromReader(RenderConletRequestBase<?> request,
1154                String conletType, String conletId, Reader contentReader) {
1155            super(conletType, conletId);
1156            // Start to prepare the content immediately and concurrently.
1157            content = request.processedBy().map(pby -> pby.executorService())
1158                .orElse(Components.defaultExecutorService()).submit(() -> {
1159                    StringWriter content = new StringWriter();
1160                    CharBuffer buffer = CharBuffer.allocate(8192);
1161                    try (Reader rdr = new BufferedReader(contentReader)) {
1162                        while (true) {
1163                            if (rdr.read(buffer) < 0) {
1164                                break;
1165                            }
1166                            buffer.flip();
1167                            content.append(buffer);
1168                            buffer.clear();
1169                        }
1170                    } catch (IOException e) {
1171                        throw new IllegalStateException(e);
1172                    }
1173                    return content.toString();
1174                });
1175
1176        }
1177
1178        /**
1179         * Content.
1180         *
1181         * @return the future
1182         */
1183        @Override
1184        public Future<String> content() {
1185            return content;
1186        }
