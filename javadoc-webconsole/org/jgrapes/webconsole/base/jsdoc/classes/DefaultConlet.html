Class DefaultConlet

A default implementation ov Conlet that assumes that the conlet representation in the DOM is an element with class conlet and a dataset attribute conlet-id. Different kinds of conlet representationa are distinguished by additional classes conlet-preview, conlet-view and conlet-content.
